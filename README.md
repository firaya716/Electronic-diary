# Electronic-diary
Веб приложение для автоматизаций управления учебными записями в образотвательных учреждениях.
<img width="493" height="269" alt="image" src="https://github.com/user-attachments/assets/8907e7f0-0d63-45f8-8aa3-ad4d0eec08c0" />
# Основной функционал
- Добавление и просмотр список учащихся
- Ведение электронного журнала успеваемости
- подсчет двоек в четверти
